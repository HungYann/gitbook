### 刷题步骤 
在刷的过程中，我会按照如下规则记录自己的刷题步骤：

- 1.原题目（简单，中等，困难）
- 2.自己的解法第一遍
- 3.网上好的解法
- 4.自己可以改进的地方
- 5.进一步精简优化自己的代码直至代码简无可简
- 6.获得的思考
